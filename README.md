# Pulse Of Raga

A Carnatic raga explorer — a small installable web app (PWA). Browse hero
ragas with their scales (arohanam / avarohanam), rasa and mood, master
recordings, and ready-made Suno fusion seeds.

**Live:** https://stashgoblin.github.io/pulse-of-raga/

A static site — no build step. Raga data is derived from the Carnatic archive
at [shivkumar.org](https://www.shivkumar.org/music/); audio is streamed from
that archive. Generated by the MusicMaker project.
